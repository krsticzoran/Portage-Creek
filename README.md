# Portage Creek Web Designs

High-performance custom website built with Next.js 15 and Tailwind CSS 4.

## 🚀 Performance

- **Lighthouse Score:** 100/100
- **Fully Responsive:** Mobile, tablet, and desktop
- **Dark Mode:** Theme toggle with persistent storage
- **Optimized Images:** Lazy loading and responsive sizing

## 🛠️ Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS 4
- **Theme:** next-themes
- **Deployment:** Netlify
- **Language:** JavaScript/React

## 📦 Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/krsticzoran/Portage-Creek.git

# Navigate to project directory
cd Portage-Creek

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## 🏗️ Build

```bash
# Create production build
npm run build

# Start production server
npm start
```
